Choisir les transitions 2D - les transitions 3D

Placez votre souris sur les noms des transitions pour voir leurs effets.


Indiquez les transitions voulues dans les propriétés transition2d et transition3d

- soit par leurs numéros séparés par des virgules,
- soit par le mot-clé all si vous souhaitez que le slider choisisse les transitions au hasard.

Veuillez noter que seuls les navigateurs internet récents sont compatibles avec les transitions 3D css3. Vous pouvez utiliser les transitions 3D avec les dernières versions de Internet Exlorer, Chrome (sauf sur Windows XP), Firefox, Safari, bientôt avec Opera, et bien sûr avec les navigateurs mobile compatibles. Pour tous les autres navigateurs, le Slider utilisera automatiquement les transitions 2D (choisies ou, à défaut, au hasard).
